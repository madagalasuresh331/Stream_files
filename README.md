# MaTtErS...AnAlYzE

MaTtErS...AnAlYzE is a Streamlit-based AI Data Analysis Agent used to upload CSV or Excel files, clean data, analyze data, create pivot tables, generate graphs, and download final reports.

## Features

- Upload CSV and Excel files
- Show dataset preview
- Display data info, describe, and correlation
- Fill null values using mean or median
- Fill text null values as Unknown
- Remove duplicate rows
- Create pivot analysis
- Generate bar, line, pie, and combo charts
- Download cleaned dataset
- Download final analysis report

## Tools Used

Python, Streamlit, Pandas, Plotly

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
